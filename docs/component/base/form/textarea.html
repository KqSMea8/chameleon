textarea

textarea

多行输入框

属性

属性名 类型 必填 默认值 说明
value String 否 多行输入框的初始内容
type String 否 "text" 多行输入框的类型
placeholder String 否 提示用户输入的内容
disabled Boolean 否 false 是否禁用
focus Boolean 否 false 获取焦点
maxlength Number 否 140 最大输入长度
return-key-type String 否 "done" 设置键盘右下角按钮的文字
placer-holder-color String 否 “#666” 指定placeholder的颜色（web端不支持）
input-style String 否 自定义样式,样式字段采用驼峰或者短横线命名
rows Number 否 2 text-area行数，weex、web端生效
c-bind:inputevent EventHandle 否 键盘输入时触发
返回事件对象：
event.type="inputEvent",
event.detail={value}
c-bind:confirmevent EventHandle 否 点击完成按钮时触发
返回事件对象：
event.type="confirmEvent",
event.detail
c-bind:focusevent EventHandle 否 输入框获取焦点时触发
返回事件对象：
event.type="focusEvent",
event.detail
c-bind:blurevent EventHandle 否 输入框失去焦点时触发
返回事件对象：
event.type="blurEvent",
event.detail
type 的有效值：
值 说明
text 文本类型的输入
password 密码类型的输入
number 数字类型的输入
return-key-type 的有效值：
值 说明
done 右下角按钮为“完成”
search 右下角按钮为“搜索”
next 右下角按钮为“下一个”
go 右下角按钮为“前往”

}
